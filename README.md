# NodeJS REST API
> Membangun RESTful API berbasis NodeJS

### Prerequisites

Prasyaratnya yaitu:

* Download dan Install NodeJS
* Download dan Install NPM

### Installation

Silahkan clone repo ini dan lakukan:

```
# clone repo
$ git clone https://github.com/rifandani/NodeJS-REST-API.git

# masuk ke app's directory
$ cd my-project

# install app's dependencies
$ npm install
```

## Usage

Jalankan aplikasi dengan command:

```
# jalankan aplikasi mode development dengan package nodemon
$ npm run dev

```
